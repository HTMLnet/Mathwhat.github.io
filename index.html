MathWhat Language

MathWhat is an esoteric sub-language that relies entirely on JavaScript as its execution base. It transforms normal JavaScript source code into a strange symbolic representation using only six operators: +, -, (, ), ×, and ÷. Each character of code is converted into binary, which is then expressed through these symbols, allowing MathWhat to rebuild the original JavaScript logic when interpreted. The language is intentionally impractical, pushing the limits of readability and minimalism. It is designed to demonstrate how JavaScript can be abused, encoded, and reconstructed in unconventional ways. This makes MathWhat a playful, yet technically functional tool to explore encoding, interpretation, and the boundaries of language design in web environments.

MathWhat Operations
  • + → Push 1
  • - → Push 0
  • ( → Start char build
  • ) → End char build (collects bits, makes a char)
  • × → Append built char to program
  • ÷ → Clear buffer
